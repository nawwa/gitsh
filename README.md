Simple git script for epitech or git users.

Usage :

./gitsh
	   -c(clone) [login] <repository>
           -p(push) <commit>
           -g(pull)
	   -m(merge) <commit>
           -i(init)
           -l(blih list)
           -s(acl) <repo> <user> [acl]
           -gs(getacl) 
	   ...
